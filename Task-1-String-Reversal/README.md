## Task 1: String Reversal

### Description
This program takes a string as input and reverses it using a two-pointer approach.
The logic swaps characters from the beginning and end of the string until the
entire string is reversed.

### Concepts Used
- Strings
- Loops
- Two-pointer technique
- Basic input/output

### How to Compile and Run
```bash
g++ stringReversal.cpp -o stringReversal
./stringReversal
