## Task 4: Guessing Game

### Description
This program implements a simple number guessing game. The program generates a
random number and prompts the user to guess it, providing hints until the correct
number is guessed.

### Concepts Used
- Random number generation
- Loops
- Conditional statements
- User input/output

### How to Compile and Run
```bash
g++ guessing_game.cpp -o guessing_game
./guessing_game
